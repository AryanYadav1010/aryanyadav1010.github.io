# Retro Gamified Developer Portfolio 

A vintage, retro game-inspired developer portfolio website. Built with pure HTML, CSS, and Vanilla JavaScript, this portfolio features dynamic CSS-drawn pixel art, parallax scrolling backgrounds, and a gamified interface.

## 🔗 Live Site
[View Live Portfolio](https://AryanYadav1010.github.io/)

## ✨ Features

- **Pure CSS Pixel Art:** Intricate, animated pixel art scenes (Ecommerce, Dashboard, Chat, and Travel monitors) built entirely with CSS—no image assets required.
- **Parallax Scrolling:** Dynamic, multi-layered background scrolling effect that responds to user navigation.
- **Retro Aesthetic:** Custom pixel block fonts, vintage color palettes, and arcade-style button interactions.
- **Fully Responsive:** Adapts seamlessly specifically for both desktop and mobile viewing with a custom mobile sliding menu.
- **Interactive Elements:** Hover animations, glowing text effects, and game-themed decorative elements (coins, pipes, blocks).

## 🗂️ Sections

- **Projects:** Showcasing development projects.
- **Experience:** Professional work history.
- **Education:** Academic background and degrees.
- **Skills:** Language and technical tool proficiencies.
- **Contact:** Final "Course Clear" screen with contact details.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and custom data attributes.
- **CSS3:** Advanced styling, CSS variables, flexbox, CSS grid, keyframe animations, typography, and pure CSS drawings.
- **Vanilla JavaScript:** DOM manipulation, smooth scrolling, intersection observers for animations, and mobile menu toggling.

## 🚀 Running Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/AryanYadav1010/aryanyadav.github.io.git
   ```
2. Navigate into the project directory:
   ```bash
   cd aryanyadav.github.io
   ```
3. Start a local server. If you have Python installed, you can easily use:
   ```bash
   python3 -m http.server 8000
   ```
4. Open your browser and navigate to `http://localhost:8000`
